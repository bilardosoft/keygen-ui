# [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs) Event logsBeta

Event logs are automatically populated as events are made on your Keygen
account. Event logs can be used for auditing and debugging purposes.

**Event logs are available for all Keygen Cloud customers, as well as**
**for all Keygen EE customers.** Event logs are not available in
Keygen CE. If you're running Keygen CE and would like to try event
logs, [reach out](mailto:sales@keygen.sh) for a 30-day no-strings-attached
trial of Keygen EE.

**This feature is currently in beta. Please [reach out](mailto:support@keygen.sh)**
**to gain access to the beta program, or to report any feedback or issues.**
All functionality is subject to change until the beta designation
is removed from the feature.

## [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object) The event log object

Below you will find the various attributes for the event log resource.

Event log retention will depend on your subscription tier. By default, our Std tiers
have a retention period of 3 days, and our Ent tiers have a retention period of 30
days. If you're interested in longer retention, [reach out](mailto:sales@keygen.sh).

### [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object-attrs) Attributes

- #### [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object-attrs-event) data.attributes.event

stringread only



The event type.

- #### [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object-attrs-metadata) data.attributes.metadata

object<string, any>read only



Object containing event [metadata](https://keygen.sh/docs/api/metadata/).



Depending on the event type, this may include a license validation code, a diff of the resource's updated attributes (new, old), what version an upgrade was from, etc. All metadata is subject to change.

- #### [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object-attrs-created) data.attributes.created

timestamp (iso8601)read only



When the event was triggered.

- #### [_link_](https://keygen.sh/docs/api/event-logs/\#event-logs-object-attrs-updated) data.attributes.updated

timestamp (iso8601)read only



When the event log was last updated.
