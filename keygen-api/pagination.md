# [_link_](https://keygen.sh/docs/api/pagination/\#pagination) Pagination

All top-level API resources have support for bulk fetches via "list" API
methods. For instance you can [list users](https://keygen.sh/docs/api/users/#users-list), [list licenses](https://keygen.sh/docs/api/licenses/#licenses-list),
and [list machines](https://keygen.sh/docs/api/machines/#machines-list). These list API methods share a common
format, taking an optional `page` query parameter.

Our API utilizes a page-based strategy via the `page[size]` and `page[number]`
parameters. All results are returned in reverse chronological order.

### [_link_](https://keygen.sh/docs/api/pagination/\#pagination-object-query) Query parameters

- #### [_link_](https://keygen.sh/docs/api/pagination/\#pagination-object-query-page) page

object<string, integer>



Object containing page `size` and page `number`.

- #### [_link_](https://keygen.sh/docs/api/pagination/\#pagination-object-query-page[size]) page\[size\]

integer



The page size. Must be a number between 1 and 100.

- #### [_link_](https://keygen.sh/docs/api/pagination/\#pagination-object-query-page[number]) page\[number\]

integer



The page number.
